# **App Name**: ImmobManager

## Core Features:

- Dashboard KPIs: Display key performance indicators, including total sales, commissions, monthly sales, and conversion rates.
- Sales Visualization: Provide a bar graph illustrating sales per constructor.
- Action Items: List clients who are stalled in the credit analysis phase for an extended period to draw attention for potential action.
- Sales Module: Implement sales management functionality with both Kanban (board) and list views.
- Kanban Status Columns: Organize the Kanban board with columns representing each stage of the sales pipeline: New Registration, Credit Analysis, Approved / Awaiting Unit, Pending Signature, Completed Sale, Sale Dropped / Cancelled.
- Sales Card Details: Display client name, project name, down payment value, and sales agent name on each card in the Kanban view; differentiate builders with colored badges (e.g., Tenda = Blue, Vasco = Orange).

## Style Guidelines:

- Primary color: Navy blue (#2E4A62) for a professional and trustworthy look.
- Background color: Light gray (#F5F5F5) to provide a clean, uncluttered backdrop.
- Accent color: Green (#4CAF50) to highlight monetary values and successful actions, representing money and positive outcomes.
- Body and headline font: 'Inter', a sans-serif font for a modern, objective feel.
- Lucide React icons for a clean, consistent, and modern aesthetic throughout the application.
- Responsive layout design that adapts to different screen sizes, ensuring a consistent user experience on both desktop and mobile devices.
- A sidebar on the left for navigation between Dashboard, Sales, Agents, and Finance modules for easy access and improved user experience.
